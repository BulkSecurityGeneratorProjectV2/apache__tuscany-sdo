Tuscany SDO Java M2 Sample Program Distribution
===============================================

This is a distribution of the sample programs for the Tuscany SDO Java implementation.
The distribution contains the sample source code,  the compiled code,  all the required dependencies
to run the samples,  and javadoc to describe the contents of the distribution.

The root of the javadoc for the samples is in the file javadoc/index.html.  Look at the documentation to
decide which sample to run and for instructions on how to run them.
